# webcmd-plugin-techcrunch

Read the latest TechCrunch news through WebCMD using TechCrunch's public RSS feed. No login is required.

## Install

```bash
webcmd plugin install github:agentrhq/webcmd/plugins/techcrunch
```

## Commands

| Command | Description |
|---------|-------------|
| `techcrunch latest` | List the latest TechCrunch stories |

## Examples

```bash
webcmd techcrunch latest
webcmd techcrunch latest --limit 10
```
